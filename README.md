# Course Management API

## 📌 Description

This project is an ASP.NET Core Web API for managing students, courses, and instructors.

## ⚙️ Technologies Used

* ASP.NET Core Web API
* Entity Framework Core
* SQL Server
* JWT Authentication

## 🚀 Features

* CRUD operations for Students and Courses
* Entity relationships (1-1, 1-M, M-M)
* DTO validation
* Dependency Injection
* JWT Authentication & Authorization

## 🔐 Authentication

JWT tokens are used for securing endpoints.
Users must login and include the token in requests.

## ▶️ How to Run

1. Run:
   dotnet run
2. Open Swagger:
   http://localhost:5063/swagger
3. Login using:
   username: omar
   password: 1234
